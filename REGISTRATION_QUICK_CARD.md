## 📋 Two Ways to Register

### Method 1: Quick Start (⚡ FASTEST)
1. Go to: https://density-bbe08.web.app
2. Click: "⚡ Quick Start" tab
3. Click: Any ticket button (e.g., DX-005491)
4. ✅ Done! Dashboard loads instantly

### Method 2: Manual Registration (📝)
1. Go to: https://density-bbe08.web.app
2. Stay on: "📝 Register" tab
3. Fill:
   - **Ticket ID:** DX-XXXXXX (e.g., DX-005491)
   - **Name:** Your name
   - **Phone:** 10+ digit number
4. Click: "✅ Register & Enter"
5. ✅ Done! Dashboard appears

## 📍 What You Get

### Left Side (70%)
- 🗺️ Interactive Leaflet map
- 👤 Blue dots = Users
- 🔴🟡🟢 Color-coded clusters
  - 🟢 Green: 1-5 users
  - 🟡 Yellow: 6-20 users
  - 🔴 Red: 21+ users (ALERT!)
- 📊 200 live users visible
- 🔄 Updates every 5 seconds

### Right Side (30%)
- 🚨 Alert Panel (flashing for dense clusters)
- 📊 Stats: Users, clusters, density %
- 🔍 Cluster details grid
- 🕐 Refresh control (2-30 seconds)
- 🚪 Logout button (top right)
